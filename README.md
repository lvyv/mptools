**路径规划微服务**
=====================
本项目借用V2V的多进程调度框架，实现对本地的MAPF算法包的调度，并以微服务的方式进行发布，供前端调用。

一、快速体验
--------
### 预装环境
- Windows10系统。
- 已经安装Miniconda3。
- 创建python 3.7.8虚拟环境，安装依赖包（requirements.txt）。

### 运行步骤
Windows10环境下代码演示如下。

1.打开Pycharm，用Pycharm打开根目录创建工程，并如下配置运行参数。
![img.png](docs\images\envi.png)
2.运行工程，访问[本地服务](http://127.0.0.1:7080/docs)。
